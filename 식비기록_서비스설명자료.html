1. 서비스 소개

식비기록은 가족 또는 커플이 함께 사용하는 식비 전용 가계부 앱입니다. 구글 로그인 한 번이면 바로 시작할 수 있으며, 초대 코드를 공유해 같은 그룹으로 묶여 식비를 함께 기록하고 관리합니다.

핵심 특징

  • 구글 계정으로 간편 로그인 (별도 회원가입 불필요)
  • 초대 코드 한 줄로 가족/커플 그룹 연결
  • 누가 입력했는지 구분하지 않는 공동 장부 컨셉
  • 카테고리별 지출 분석 및 월별 통계 차트
  • 영수증 사진 첨부 기능
  • 상세 품목 입력 및 구매 내역 펼쳐보기
  • 품목 클릭 시 과거 구매 내역 조회 (가격 추이, 평균/최저/최고가)
  • 모바일 최적화 UI (하단 탭 네비게이션)

5. 구현 현황 및 로드맵

완료된 기능

  • 완료 구글 OAuth 로그인
  • 완료 그룹 생성 / 초대코드 참여
  • 완료 지출 입력 (금액, 카테고리, 날짜, 메모, 사진)
  • 완료 지출 내역 (날짜 그룹, 필터, 삭제)
  • 완료 상세 품목 펼쳐보기 (품목별 금액 조회)
  • 완료 대시보드 (이번 달 합계, 카테고리 요약)
  • 완료 통계 (파이 차트, 월별 바 차트)
  • 완료 설정 (프로필, 초대코드 복사, 로그아웃)
  • 완료 영수증 사진 업로드
  • 완료 품목 과거 구매 내역 조회 (가격 추이, 통계)

예정된 기능 (로드맵)

  • 예정 지출 수정 기능
  • 예정 카테고리 추가/편집 UI
  • 예정 PWA 설정 (홈 화면 추가)
  • 예정 영수증 AI 인식 (OCR)
  • 예정 반응형 데스크톱 레이아웃
  • 예정 중복 제출 방지

데이터 보안

모든 데이터 테이블에 행 수준 보안(Row Level Security)이 적용되어 있습니다.
로그인한 사용자가 속한 그룹의 데이터에만 접근할 수 있으며, 다른 그룹의 데이터는 열람/수정이 불가합니다.
